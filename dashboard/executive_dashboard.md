## 📊 Power BI Interactive Executive Dashboard

To demonstrate the responsive cross-filtering capabilities of the Gold Layer star schema, the visual layouts below showcase the application in both its default aggregate state and dynamically filtered for regional hotspot analysis.

### 🌐 View 1: Default Network Overview (Unfiltered)
* **Operational Scope:** Captures all monitored domestic healthcare facilities across all 30-day readmission clinical measures. 
* **Key Visuals:** Global KPI cards, nationwide high-risk metric summaries, and complete operational roster details.

![Power BI Dashboard - Unfiltered Overview](dashboard/dashboard_unfiltered.png)
---

### 🗺️ View 2: Regional Hotspot Analysis (Florida Selected)
* **Operational Scope:** Dynamically sliced via the geography filter pane to isolate regional performance markers.
* **Key Visuals:** Notice how the KPI blocks, high-risk percentages, and the detailed hospital roster automatically cross-filter to show only facilities operating within the Florida state boundary boundaries.

![Power BI Dashboard - Florida Filtered View](dashboard/dashboard_filtered.png)
